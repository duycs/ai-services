# AI API
- NodeJS .TS
- Google Gemini 
- Openai 

## Environment

```sh
npm install
```

## Keys
- [https://makersuite.google.com/app/apikey]
- [https://platform.openai.com/]

Update your `.env` file with your GEMINI_API_KEY, OPENAI_API_KEY

## Build

```sh
npm run build
```

## Run

```sh
npm run start
```
